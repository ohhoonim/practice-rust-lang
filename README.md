# 자바 개발자를 위한 Rust

## Rust 설치

검색하면 다 나옴.

## 프로젝트 초기화 

폴더를 만들고 폴더 안쪽에서, 

```sh
$ mkdir [폴더명]
$ cd [폴더명]
$ cargo init
```

`src` 폴더, `.gitignore`, `Cargo.toml` 파일이 생성된다.
`cargo new` 를 사용하면 프로젝트 명으로 폴더까지 다 만들어주므로 선택은 네가 해.


## 테스트코드 작성

- 단위 테스트: `#[cfg(test)]` 모듈과 `#[test]` 속성을 사용해 작성
- 통합 테스트: 프로젝트 루트의 `tests/` 디렉토리에 독립된 `.rs` 파일로 작성
