# 함수 정의, 매개변수 및 반환 값 처리

- **요약 목록**
    - **함수 정의**: `fn` 키워드를 사용하며, 스네이크 케이스(`snake_case`) 형태로 이름을 명명합니다.
    - **매개변수**: 함수의 입력값은 반드시 `변수명: 타입` 형태로 명시해야 합니다.
    - **반환 값**: 화살표(`>`) 뒤에 타입을 명시하며, `return` 키워드를 사용하거나 함수 마지막 줄에 세미콜론 없는 표현식(Expression)을 사용하여 반환합니다.

## 함수 정의와 매개변수 (Parameters)

Rust에서 함수는 `fn` 키워드로 시작하며, 함수 이름과 변수 이름에는 **스네이크 케이스(snake_case)** 스타일을 사용하는 것이 관례입니다.

함수를 정의할 때 **각 매개변수의 타입은 반드시 명시**해야 합니다. 이는 컴파일러가 함수가 어떻게 사용되는지 추론하기 위해 코드의 다른 부분을 뒤지지 않아도 되도록 설계된 의도적인 선택입니다.

```rust
fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("측정값: {value}{unit_label}");
}
```

## 반환 값 처리 (Return Values)

함수가 값을 반환할 때는 매개변수 소괄호 `()` 뒤에 **화살표(`->`)와 반환할 타입을 명시**합니다. Rust에서 값을 반환하는 방법은 두 가지가 있습니다.

### 1. 표현식(Expression) 반환 (권장 방식)

Rust는 식(Expression) 기반 언어입니다. 함수의 마지막 줄에 **세미콜론(`;`)을 붙이지 않은 표현식**을 두면, 그 값이 자동으로 함수의 반환 값이 됩니다.

```rust
fn five() -> i32 {
    5
}
```

- `5` 뒤에 세미콜론이 없으므로 이 함수는 `5`를 반환합니다. 만약 `5;`라고 적으면 구문(Statement)이 되어 아무것도 반환하지 않으므로 컴파일 에러가 발생합니다.

### 2. `return` 키워드 반환

함수 중간에 조건에 따라 조기 종료하고 값을 반환해야 할 때는 `return` 키워드를 사용할 수 있습니다.

```rust
fn plus_one(x: i32) -> i32 {
    if x > 100 {
        return x; // 조기 반환 시에는 return과 세미콜론 사용
    }
    x + 1 // 마지막 줄 표현식 반환
}
```
