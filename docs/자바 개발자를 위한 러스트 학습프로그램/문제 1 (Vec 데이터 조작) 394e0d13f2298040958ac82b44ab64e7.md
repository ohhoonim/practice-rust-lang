# 문제 1 (Vec 데이터 조작)

## [문제 1] Vec의 동적 제어와 소유권 이해

### 설명

자바의 `List<String>` 처럼 가변 배열인 `Vec<String>`을 다루는 훈련입니다.

주어진 작업 목록(Vector)에 새로운 작업을 추가하고, 특정 인덱스의 작업을 꺼내어 소유권을 완전히 이동시키는 함수들을 완성하세요.

- `add_task`: 벡터에 가변 참조(`&mut`)로 접근하여 새로운 작업을 끝에 추가합니다.
- `complete_task`: 벡터에서 특정 인덱스의 요소를 완전히 꺼내어(`remove`) 반환합니다. (소유권 이동)

    fn test_vec_operations() {
        let mut todo = TodoList::new();

        todo.add_task("Clean room".to_string());
        todo.add_task("Write code".to_string());

        assert_eq!(todo.tasks.len(), 2);

        // 0번 인덱스의 "Clean room"을 제거하며 소유권을 가져옴
        let completed = todo.complete_task(0);
