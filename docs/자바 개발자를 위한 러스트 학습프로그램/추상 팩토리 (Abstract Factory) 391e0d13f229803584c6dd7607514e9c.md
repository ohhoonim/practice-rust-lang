# 추상 팩토리 (Abstract Factory)

- **목표:** 연관된 제품들의 군(Family)을 생성하기 위한 인터페이스를 제공하는 추상 팩토리 패턴을 구현합니다.
- **조건:** 팩토리 메서드가 단일 제품의 생성에 집중한다면, 추상 팩토리는 관련된 여러 제품(예: UI 테마의 버튼과 체크박스)을 일관성 있게 생성하는 테마형 구조를 정의합니다. Rust의 `trait`와 `Box<dyn Trait>`를 활용합니다.

### [연습 문제] 추상 팩토리 패턴 구현

상반된 UI 테마(맥 스타일과 윈도우 스타일)에 맞춰 버튼(Button)과 체크박스(Checkbox)를 일관되게 생성하는 GUI 팩토리 시스템을 구현하세요.

1. 제품 인터페이스: `Button`과 `Checkbox` 트레이트는 각각 `paint` 메서드를 가집니다.
2. 추상 팩토리 인터페이스: `GuiFactory` 트레이트는 `create_button`과 `create_checkbox` 메서드를 가집니다.
3. 구체적인 팩토리: `MacFactory`와 `WinFactory` 구조체는 각각 `GuiFactory`를 구현하며, 자신의 테마에 맞는 제품 쌍(`MacButton`/`MacCheckbox`, `WinButton`/`WinCheckbox`)을 생성합니다.

```rust
// 1. 제품(Product) 추상화 및 구체적인 구현체들
trait Button {
    fn paint(&self) -> String;
}

trait Checkbox {
    fn paint(&self) -> String;
}

struct MacButton;
impl Button for MacButton {
    fn paint(&self) -> String { "Mac Button".to_string() }
}

struct MacCheckbox;
impl Checkbox for MacCheckbox {
    fn paint(&self) -> String { "Mac Checkbox".to_string() }
}

struct WinButton;
impl Button for WinButton {
    fn paint(&self) -> String { "Win Button".to_string() }
}

struct WinCheckbox;
impl Checkbox for WinCheckbox {
    fn paint(&self) -> String { "Win Checkbox".to_string() }
}

// 2. 추상 팩토리(Abstract Factory) 인터페이스
trait GuiFactory {
    fn create_button(&self) -> Box<dyn Button>;
    fn create_checkbox(&self) -> Box<dyn Checkbox>;
}

// 3. 구체적인 팩토리(Concrete Factory) 구현
struct MacFactory;
impl GuiFactory for MacFactory {
    // 이 부분을 구현하세요.
}

struct WinFactory;
impl GuiFactory for WinFactory {
    // 이 부분을 구현하세요.
}

#[cfg(test)]
mod tests {
    use super::*;

    // 팩토리를 소비하는 클라이언트 코드 예시
    fn client_code(factory: &dyn GuiFactory) -> (String, String) {
        let button = factory.create_button();
        let checkbox = factory.create_checkbox();
        (button.paint(), checkbox.paint())
    }

    #[test]
    fn test_abstract_factory() {
        let mac_factory = MacFactory;
        let win_factory = WinFactory;

        assert_eq!(client_code(&mac_factory), ("Mac Button".to_string(), "Mac Checkbox".to_string()));
        assert_eq!(client_code(&win_factory), ("Win Button".to_string(), "Win Checkbox".to_string()));
    }
}
```
