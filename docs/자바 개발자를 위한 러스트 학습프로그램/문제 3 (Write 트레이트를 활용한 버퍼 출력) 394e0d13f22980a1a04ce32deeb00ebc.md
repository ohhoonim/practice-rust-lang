# 문제 3 (Write 트레이트를 활용한 버퍼 출력)

## [문제 3] Write 트레이트를 활용한 공통 출력 다형성

### 설명

자바에서 `void writeData(OutputStream out)` 인터페이스를 정의하면 파일 출력(`FileOutputStream`)이든 네트워크 출력(`SocketOutputStream`)이든 가리지 않고 전달할 수 있습니다.

Rust에서는 **`std::io::Write`** 트레이트가 이 역할을 담당합니다.

주어진 문자열 슬라이스 배열을 줄바꿈과 함께 출력 스트림에 쓰는 함수를 완성하세요.

- 매개변수 `writer`는 `&mut W` 형태로 전달되며, `W: Write` 제네릭 제약조건을 가집니다.
- `writeln!(writer, "{}", item)` 매크로를 사용하면 편리하게 줄바꿈과 함께 데이터를 밀어 넣을 수 있습니다.

### 템플릿 코드

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_write_trait() {
        // Vec<u8> 구조체는 Write 트레이트를 구현하고 있으므로 메모리 버퍼 스트림 역할을 수행합니다.
        let mut buffer = Vec::new();

        let result_string = String::from_utf8(buffer).unwrap();
        assert_eq!(result_string, "Spring\nBoot\n4\n");
