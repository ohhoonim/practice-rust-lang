# 전략 (Strategy)

- **목표:** 실행 중에 알고리즘(전략)을 동적으로 교체할 수 있도록, 공통 인터페이스를 정의하고 이를 구체적인 클래스들로 캡슐화하는 전략 패턴을 구현합니다.
- **조건:** Rust에서는 정적 디스패치(`impl Strategy` 또는 제네릭 사용) 방식과 동적 디스패치(`Box<dyn Strategy>` 사용) 방식을 모두 쓸 수 있습니다. 이번 연습에서는 런타임에 전략을 자유롭게 유연하게 변경할 수 있도록 트레이트 객체를 사용한 동적 디스패치 방식을 사용합니다.

### [연습 문제] 전략 패턴 구현

결제 시스템에서 금액을 계산할 때 일반 결제 방식과 할인가가 적용되는 결제 방식을 동적으로 변경할 수 있는 `ShoppingCart` 시스템을 구현하세요.

1. **Strategy 인터페이스:** `PaymentStrategy` 트레이트는 가격(`price`)을 입력받아 최종 청구 금액을 반환하는 `calculate_price` 메서드를 가집니다.
2. **구체적인 전략들:** * `NormalStrategy`: 입력받은 금액을 그대로 반환합니다.
    - `DiscountStrategy`: 입력받은 금액에서 10%를 할인한 금액을 반환합니다.
3. **Context 객체:** `ShoppingCart` 구조체는 내부에 선택된 전략인 `Box<dyn PaymentStrategy>`를 가집니다. `checkout` 메서드는 현재 설정된 전략을 사용해 최종 금액을 계산합니다.

```rust
// 1. 전략 인터페이스 (Strategy)
trait PaymentStrategy {
    fn calculate_price(&self, price: u32) -> u32;
}

// 2. 구체적인 전략 1 (Concrete Strategy - 일반 결제)
struct NormalStrategy;
impl PaymentStrategy for NormalStrategy {
    fn calculate_price(&self, price: u32) -> u32 {
        price
    }
}

// 3. 구체적인 전략 2 (Concrete Strategy - 할인 결제)
struct DiscountStrategy;
impl PaymentStrategy for DiscountStrategy {
    // calculate_price를 구현하세요.
    // price에서 10%를 할인한 금액(즉, price * 90 / 100)을 계산하여 반환합니다.
    fn calculate_price(&self, price: u32) -> u32 {
        // 이 부분을 구현하세요.
    }
}

// 4. 문맥 객체 (Context)
struct ShoppingCart {
    strategy: Box<dyn PaymentStrategy>,
}

impl ShoppingCart {
    fn new(strategy: Box<dyn PaymentStrategy>) -> Self {
        ShoppingCart { strategy }
    }

    // 전략을 동적으로 변경하는 메서드
    fn set_strategy(&mut self, strategy: Box<dyn PaymentStrategy>) {
        self.strategy = strategy;
    }

    // 현재 전략을 사용하여 최종 금액을 반환하는 메서드
    fn checkout(&self, price: u32) -> u32 {
        // 이 부분을 구현하세요.
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_strategy_pattern() {
        // 1. 기본 전략(Normal)으로 카트 생성
        let mut cart = ShoppingCart::new(Box::new(NormalStrategy));
        assert_eq!(cart.checkout(10000), 10000);

        // 2. 런타임에 할인 전략(Discount)으로 교체
        cart.set_strategy(Box::new(DiscountStrategy));
        assert_eq!(cart.checkout(10000), 9000);
    }
}
```
