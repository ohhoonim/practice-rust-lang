# 기본 타입: 정수, 부동 소수점, 부울, 문자(char)

- **요약 목록**
    - **정수형**: 부호 유무(`i`/`u`)와 비트 크기(`8`~`128`, `size`)별로 세분화되어 있으며, 기본값은 `i32`입니다.
    - **부동 소수점형**: 정밀도에 따라 `f32`, `f64`가 있으며, 기본값은 `f64`입니다.
    - **부울형**: `true`와 `false` 두 가지 값만 가집니다.
    - **문자형(`char`)**: 4바이트 크기의 Unicode 스칼라 값으로, 유니코드 전체(이모지 등)를 표현할 수 있으며 작은따옴표(`'`)를 사용합니다.

## 정수형 (Integer Types)

소수점이 없는 숫자 타입입니다. 부호가 있는 정수(`i`)와 부호가 없는 정수(`u`)로 나뉘며, 메모리 크기(비트)에 따라 분류됩니다.

| **크기** | **부호 있음 (Signed)** | **부호 없음 (Unsigned)** |
| --- | --- | --- |
| 8-bit | `i8` | `u8` |
| 16-bit | `i16` | `u16` |
| 32-bit | `i32` (기본값) | `u32` |
| 64-bit | `i64` | `u64` |
| 128-bit | `i128` | `u128` |
| arch (컴퓨터 아키텍처 의존) | `isize` | `usize` |
- `isize`와 `usize`는 프로그램이 실행되는 컴퓨터의 아키텍처(64비트 시스템이면 64비트)에 따라 크기가 결정되며, 주로 컬렉션의 인덱스로 사용됩니다.

## 부동 소수점형 (Floating-Point Types)

소수점이 있는 숫자 타입입니다. IEEE-754 표준을 따르며 두 가지 크기가 존재합니다.

```rust
let x = 2.0; // f64 (기본값)
let y: f32 = 3.0; // f32
```

- `f32`: 단정밀도(Single precision) 부동 소수점
- `f64`: 배정밀도(Double precision) 부동 소수점 (최신 CPU에서는 속도가 거의 비슷하면서도 더 정밀하므로 기본값으로 사용됨)

## 부울형 (Boolean Type)

`true`와 `false` 두 가지 값만 가질 수 있으며, 크기는 1바이트입니다.

```rust
let t = true;
let f: bool = false; // 명시적 타입 선언
```

## 문자형 (Character Type)

가장 기본적인 알파벳, 숫자, 문자를 표현하는 타입입니다.

```rust
let c = 'z';
let z: char = 'ℤ';
let heart_eyed_cat = '😻'; // 유니코드 문자 및 이모지 가능
```

- 작은따옴표(`'`)를 사용하여 표현합니다. (큰따옴표는 문자열 리터럴)
- Rust의 `char`는 4바이트 크기의 **Unicode 스칼라 값**입니다. 따라서 영어뿐만 아니라 한글, 한자, 이모지 등 상상할 수 있는 모든 유니코드 문자를 표현할 수 있습니다.
