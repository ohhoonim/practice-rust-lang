# 프록시 (Proxy)

- **목표:** 실제 객체(Real Subject)에 대한 접근을 제어하거나 제한하기 위해 중간에서 대리자 역할을 수행하는 프록시 패턴을 구현합니다.
- **조건:** Rust에서는 실제 객체와 프록시 객체가 동일한 인터페이스(`trait`)를 구현하게 만들고, 프록시 구조체가 실제 객체를 내부에 두고 요청을 가로채어 권한 검사(Protection Proxy)나 지연 초기화(Virtual Proxy) 등을 수행하도록 설계합니다.

### [연습 문제] 프록시 패턴 구현

인터넷 서버에 접근할 때 특정 유해 사이트(웹페이지)로의 접근을 차단하는 방화벽 프록시(Protection Proxy) 시스템을 구현하세요.

1. **인터페이스:** `Internet` 트레이트는 `connect_to` 메서드를 가집니다.
2. **실제 객체:** `RealInternet` 구조체는 `Internet`을 구현하며 제한 없이 사이트에 접속한 메시지를 반환합니다.
3. **프록시 객체:** `ProxyInternet` 구조체는 내부에 `RealInternet`과 차단된 사이트 목록(`Vec<String>`)을 가집니다. `connect_to` 호출 시 목적지 사이트가 차단 목록에 있으면 접근 거부 메시지를 반환하고, 없으면 실제 객체의 `connect_to`를 호출합니다.

```rust
// 1. 공통 인터페이스 (Subject)
trait Internet {
    fn connect_to(&self, host: &str) -> String;
}

// 2. 실제 서비스 객체 (Real Subject)
struct RealInternet;
impl Internet for RealInternet {
    fn connect_to(&self, host: &str) -> String {
        format!("Connecting to {}", host)
    }
}

// 3. 접근을 제어하는 프록시 객체 (Proxy)
struct ProxyInternet {
    real_internet: RealInternet,
    banned_sites: Vec<String>,
}

impl ProxyInternet {
    fn new() -> Self {
        ProxyInternet {
            real_internet: RealInternet,
            banned_sites: vec![
                "banned-site.com".to_string(),
                "malware.io".to_string(),
            ],
        }
    }
}

// ProxyInternet에 Internet 트레이트를 구현하세요.
// connect_to 메서드는 host가 banned_sites에 포함되어 있다면 "Access Denied"를 반환하고,
// 포함되어 있지 않다면 real_internet.connect_to(host)를 호출한 결과를 반환해야 합니다.
impl Internet for ProxyInternet {
    // 이 부분을 구현하세요.
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_proxy_pattern_allowed() {
        let internet = ProxyInternet::new();

        // 허용된 사이트 접근
        assert_eq!(
            internet.connect_to("rust-lang.org"),
            "Connecting to rust-lang.org"
        );
    }

    #[test]
    fn test_proxy_pattern_denied() {
        let internet = ProxyInternet::new();

        // 차단된 사이트 접근
        assert_eq!(
            internet.connect_to("banned-site.com"),
            "Access Denied"
        );
        assert_eq!(
            internet.connect_to("malware.io"),
            "Access Denied"
        );
    }
}
```
