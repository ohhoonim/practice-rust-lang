# 반복자 (Iterator)

- **목표:** 컬렉션의 내부 구조를 노출하지 않고 요소들을 순차적으로 순회할 수 있도록 하는 반복자 패턴을 구현합니다.
- **조건:** Rust에서는 표준 라이브러리의 `Iterator` 트레이트가 제공되므로, 이를 직접 구현(`impl Iterator for ...`)하여 관용적인 반복자 패턴을 연습합니다. `next` 메서드를 정의하여 순회 로직을 캡슐화합니다.

### [연습 문제] 반복자 패턴 구현

주어진 컨테이너 내의 가짜 데이터 중 홀수(Odd) 값들만 건너뛰고 **짝수(Even) 값들만** 순차적으로 순회하는 커스텀 컬렉션의 반복자를 구현하세요.

1. `EvenIterator` 구조체는 데이터 배열에 대한 현재 인덱스(`cursor`)와 데이터 전체(`Vec<i32>`)를 가집니다.
2. `std::iter::Iterator` 트레이트를 `EvenIterator`에 구현합니다.
3. `next` 메서드가 호출될 때마다, 커서를 이동시키며 다음으로 등장하는 짝수 값을 찾아 `Some(i32)`으로 반환하고, 더 이상 짝수가 없으면 `None`을 반환합니다.

```rust
struct EvenIterator {
    cursor: usize,
    data: Vec<i32>,
}

impl EvenIterator {
    fn new(data: Vec<i32>) -> Self {
        EvenIterator { cursor: 0, data }
    }
}

// EvenIterator에 표준 라이브러리의 Iterator 트레이트를 구현하세요.
impl Iterator for EvenIterator {
    type Item = i32;

    // next 메서드를 구현하여 커서를 이동시키면서 짝수 값만 순차적으로 반환하도록 만드세요.
    fn next(&mut self) -> Option<Self::Item> {
        // 이 부분을 구현하세요.
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_iterator_pattern() {
        let numbers = vec![1, 2, 3, 4, 5, 6, 7, 8];
        let mut iterator = EvenIterator::new(numbers);

        // 반복자를 통해 짝수만 순차적으로 꺼내지는지 검증
        assert_eq!(iterator.next(), Some(2));
        assert_eq!(iterator.next(), Some(4));
        assert_eq!(iterator.next(), Some(6));
        assert_eq!(iterator.next(), Some(8));
        assert_eq!(iterator.next(), None);
    }
}
```
