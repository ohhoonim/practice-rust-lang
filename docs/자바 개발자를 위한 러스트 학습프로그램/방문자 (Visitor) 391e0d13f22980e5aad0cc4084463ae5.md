# 방문자 (Visitor)

- **목표:** 알고리즘을 객체 구조(Element)에서 분리하여, 기존 구조를 수정하지 않고도 새로운 연산(Visitor)을 동적으로 추가할 수 있는 방문자 패턴을 구현합니다.
- **조건:** Rust에서는 더블 디스패치(Double Dispatch)를 구현하기 위해 요소 트레이트에 `accept` 메서드를 정의하고, 방문자 트레이트에 각 구체적인 타입을 처리할 `visit` 메서드들을 정의합니다.

### [연습 문제] 방문자 패턴 구현

도형 구조들(원, 사각형)의 코드를 변경하지 않고, 각 도형의 이름을 출력하는 연산과 면적을 계산하는 연산을 방문자 패턴으로 구현하세요.

1. **Visitor 인터페이스:** `Visitor` 트레이트는 `visit_circle`과 `visit_rectangle` 메서드를 가집니다.
2. **Element 인터페이스:** `ShapeElement` 트레이트는 방문자를 받아들이는 `accept` 메서드를 가집니다.
3. **구체적인 방문자:** `AreaVisitor` 구조체는 각 도형을 방문하여 면적을 계산하고 그 값을 내부 가변 필드(`area`)에 누적합니다.

```rust
use std::cell::Cell;

// 1. 방문자 인터페이스 (Visitor)
trait Visitor {
    fn visit_circle(&self, circle: &Circle);
    fn visit_rectangle(&self, rectangle: &Rectangle);
}

// 2. 요소 인터페이스 (Element)
trait ShapeElement {
    fn accept(&self, visitor: &dyn Visitor);
}

// 3. 구체적인 요소들 (Concrete Elements)
struct Circle {
    radius: f64,
}
impl Circle {
    fn new(radius: f64) -> Self { Circle { radius } }
}
impl ShapeElement for Circle {
    fn accept(&self, visitor: &dyn Visitor) {
        visitor.visit_circle(self);
    }
}

struct Rectangle {
    width: f64,
    height: f64,
}
impl Rectangle {
    fn new(width: f64, height: f64) -> Self { Rectangle { width, height } }
}
impl ShapeElement for Rectangle {
    fn accept(&self, visitor: &dyn Visitor) {
        visitor.visit_rectangle(self);
    }
}

// 4. 구체적인 방문자 (Concrete Visitor - 면적 계산)
struct AreaVisitor {
    // 불변 참조 내부에서도 값을 변경할 수 있도록 Cell을 사용합니다.
    total_area: Cell<f64>,
}

impl AreaVisitor {
    fn new() -> Self {
        AreaVisitor {
            total_area: Cell::new(0.0),
        }
    }

    fn get_total_area(&self) -> f64 {
        self.total_area.get()
    }
}

// AreaVisitor에 Visitor 트레이트를 구현하세요.
// visit_circle은 원의 면적(반지름 * 반지름 * 3.14)을 계산하여 total_area에 더합니다.
// visit_rectangle은 사각형의 면적(가로 * 세로)을 계산하여 total_area에 더합니다.
// Cell의 값을 업데이트할 때는 .set()과 .get()을 활용하세요.
impl Visitor for AreaVisitor {
    // 이 부분을 구현하세요.
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_visitor_pattern() {
        // 도형 구조들의 리스트 구성 (구조 변경 없음)
        let shapes: Vec<Box<dyn ShapeElement>> = vec![
            Box::new(Circle::new(10.0)),      // 면적: 10 * 10 * 3.14 = 314.0
            Box::new(Rectangle::new(5.0, 4.0)), // 면적: 5 * 4 = 20.0
        ];

        let area_visitor = AreaVisitor::new();

        // 각 도형 구조를 순회하며 방문자 수용
        for shape in shapes.iter() {
            shape.accept(&area_visitor);
        }

        // 총 면적 합계 검증 (314.0 + 20.0 = 334.0)
        assert_eq!(area_visitor.get_total_area(), 334.0);
    }
}
```
