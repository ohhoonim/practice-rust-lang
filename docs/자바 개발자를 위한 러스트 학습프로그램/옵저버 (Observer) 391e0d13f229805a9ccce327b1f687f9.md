# 옵저버 (Observer)

- **목표:** 주체(Subject)의 상태 변화를 관찰하고 있는 여러 관찰자(Observer)들에게 변경 사항을 자동으로 통지하는 옵저버 패턴을 구현합니다.
- **조건:** Rust에서는 소유권과 빌림 검사기(Borrow Checker)의 제약으로 인해 관찰자와 주체가 상호 참조하는 구조를 만들기 어렵습니다. 여기서는 관찰자들을 스마트 포인터인 `Rc<RefCell<dyn Observer>>` 형태로 주체 내부에 벡터 컬렉션으로 저장하여 안전하게 통지하는 방식을 구현합니다.

### [연습 문제] 옵저버 패턴 구현

기상 정보 센터(`WeatherStation`)의 데이터가 업데이트될 때마다 다양한 디스플레이 장치(`CurrentConditionsDisplay`)에 자동으로 기온 정보가 전달되는 시스템을 구현하세요.

1. **Observer 인터페이스:** `Observer` 트레이트는 기온을 전달받아 갱신하는 `update` 메서드를 가집니다.
2. **Subject 객체:** `WeatherStation` 구조체는 내부에 등록된 옵저버 리스트(`Vec<Rc<RefCell<dyn Observer>>>`)를 관리하며, `register_observer`와 기온 변경 시 등록된 모든 옵저버에게 알리는 `notify_observers` 메서드를 가집니다.
3. `CurrentConditionsDisplay` 구조체는 최신 기온 값을 기록할 수 있는 가변 상태(`temperature`)를 가지며, `Observer` 트레이트를 구현합니다.

```rust
use std::cell::RefCell;
use std::rc::Rc;

// 1. 관찰자 인터페이스 (Observer)
trait Observer {
    fn update(&mut self, temperature: f32);
}

// 2. 구체적인 관찰자 객체 (Concrete Observer)
struct CurrentConditionsDisplay {
    temperature: f32,
}

impl CurrentConditionsDisplay {
    fn new() -> Self {
        CurrentConditionsDisplay { temperature: 0.0 }
    }
}

impl Observer for CurrentConditionsDisplay {
    fn update(&mut self, temperature: f32) {
        self.temperature = temperature;
    }
}

// 3. 상태 변화를 알리는 주체 객체 (Subject)
struct WeatherStation {
    observers: Vec<Rc<RefCell<dyn Observer>>>,
    temperature: f32,
}

impl WeatherStation {
    fn new() -> Self {
        WeatherStation {
            observers: Vec::new(),
            temperature: 0.0,
        }
    }

    fn register_observer(&mut self, observer: Rc<RefCell<dyn Observer>>) {
        self.observers.push(observer);
    }

    // set_measurements 메서드를 구현하세요.
    // 1. 내부의 temperature 필드를 인자로 받은 값으로 업데이트합니다.
    // 2. 등록된 모든 observers의 update 메서드를 호출하여 새로운 온도를 통지(notify)합니다.
    // Rc와 RefCell 구조이므로 borrow_mut()를 사용하여 가변 참조를 얻어와야 합니다.
    fn set_measurements(&mut self, temperature: f32) {
        // 이 부분을 구현하세요.
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_observer_pattern() {
        let mut station = WeatherStation::new();

        // 관찰자 인스턴스를 공유 및 가변 참조가 가능하도록 래핑하여 생성
        let display1 = Rc::new(RefCell::new(CurrentConditionsDisplay::new()));
        let display2 = Rc::new(RefCell::new(CurrentConditionsDisplay::new()));

        station.register_observer(Rc::clone(&display1) as Rc<RefCell<dyn Observer>>);
        station.register_observer(Rc::clone(&display2) as Rc<RefCell<dyn Observer>>);

        // 상태 변경 및 자동 통지 수행
        station.set_measurements(25.5);

        // 모든 관찰자들의 상태가 자동으로 업데이트되었는지 검증
        assert_eq!(display1.borrow().temperature, 25.5);
        assert_eq!(display2.borrow().temperature, 25.5);

        // 다시 한 번 상태 변경
        station.set_measurements(30.0);
        assert_eq!(display1.borrow().temperature, 30.0);
    }
}
```
