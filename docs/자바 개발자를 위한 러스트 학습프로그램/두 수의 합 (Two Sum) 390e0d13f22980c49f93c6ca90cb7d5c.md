# 두 수의 합 (Two Sum)

- **목표:** 정수 배열에서 두 수의 합이 타깃(Target) 값이 되는 두 원소의 인덱스를 해시맵을 사용하여 $O(N)$ 시간 복잡도로 찾습니다.
- **조건:** 데이터를 조회할 때 발생하는 `Option` 타입을 `match` 구문으로 안전하게 처리하고, 값과 인덱스를 `HashMap`에 저장하며 소유권 이동을 이해하는 훈련입니다.
