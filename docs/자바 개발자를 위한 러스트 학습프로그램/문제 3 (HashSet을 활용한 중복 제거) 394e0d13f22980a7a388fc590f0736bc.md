# 문제 3 (HashSet을 활용한 중복 제거)

## [문제 3] HashSet을 이용한 중복 없는 방문자 관리

### 설명

자바의 `HashSet`과 동일하게 중복을 허용하지 않는 고유한 값들의 집합을 관리하는 훈련입니다.

사용자의 ID가 들어올 때마다 등록하고, 최종적으로 고유한 사용자 수가 몇 명인지 반환하는 구조를 완성하세요.

- `register_user`: 이미 존재하는 ID라면 `false`를, 처음 추가된 고유 ID라면 `true`를 반환하도록 `insert` 메서드의 명세를 활용하세요.
- `total_unique_users`: 집합의 길이를 반환합니다.

        let mut manager = UserManager::new();
