# 투 포인터 기반 문제 (예: 두 수의 합 - 정렬된 상태)

- **목표:** 정렬된 정수 배열에서 두 수의 합이 타깃(Target) 값이 되는 두 원소의 인덱스를 찾는 함수를 구현합니다.
- **조건:** 배열이 이미 오름차순으로 정렬되어 있으므로, 양 끝에서 시작하는 투 포인터(Two-pointer) 접근 방식을 사용하여 $O(N)$ 시간 복잡도로 해결해야 합니다. 인덱스 범위를 벗어나지 않도록 안전하게 참조하는 구조를 연습합니다.
