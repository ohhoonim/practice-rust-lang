# 유효한 괄호 검사 (Valid Parentheses)

- **목표:** 문자열 내의 괄호 쌍이 올바르게 닫혔는지 검사하는 함수를 구현합니다.
- **조건:** `Vec<char>`를 스택(Stack)으로 활용하여 괄호를 `push`하고 `pop`합니다. 이때 `pop()` 메서드가 반환하는 `Option<char>` 타입을 안전하게 처리(패턴 매칭 또는 `if let` 등)하는 방식을 연습합니다.

### [연습 문제] 유효한 괄호 검사 (Valid Parentheses)

주어진 문자열 `s`가 괄호 `'('`, `')'`, `'{'`, `'}'`, `'['`, `']'`만으로 이루어져 있을 때, 올바른 괄호 문자열인지 판별하는 `is_valid` 함수를 완성하세요.

**올바른 괄호의 조건:**

1. 열린 괄호는 반드시 같은 타입의 닫힌 괄호로 닫혀야 합니다.
2. 열린 괄호는 올바른 순서대로 닫혀야 합니다.

```rust
fn is_valid(s: &str) -> bool {
    // 이 부분을 구현하세요.
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_is_valid() {
        assert_eq!(is_valid("()"), true);
        assert_eq!(is_valid("()[]{}"), true);
        assert_eq!(is_valid("(]"), false);
        assert_eq!(is_valid("([)]"), false);
        assert_eq!(is_valid("{[]}"), true);
    }
}
```
