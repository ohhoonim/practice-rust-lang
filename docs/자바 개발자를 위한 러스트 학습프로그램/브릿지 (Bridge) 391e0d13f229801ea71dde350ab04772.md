# 브릿지 (Bridge)

- **목표:** 추상화(Abstraction)와 구현(Implementation)을 별도의 클래스 계층 구조로 분리하여, 서로 독립적으로 확장할 수 있도록 하는 브릿지 패턴을 구현합니다.
- **조건:** Rust에서는 추상화 레이어 역할을 하는 구조체가 구현 레이어 역할을 하는 `trait` 객체를 필드로 소유(Composition)하는 방식으로 디자인을 설계합니다. 이를 통해 상속 폭발(Class Explosion) 문제를 방지합니다.

### [연습 문제] 브릿지 패턴 구현

기기(Device)들과 그 기기들을 제어하는 리모컨(Remote) 시스템을 브릿지 패턴으로 구현하세요.

1. **구현부(Implementation):** `Device` 트레이트는 `is_enabled`, `enable`, `disable` 메서드를 가집니다.
2. **추상층(Abstraction):** `Remote` 구조체는 내부에 `Box<dyn Device>`를 필드로 가져 두 계층을 연결(Bridge)하며, `toggle_power` 메서드를 제공합니다.

```rust
// 1. 구현부 (Implementation) 인터페이스 및 구체적인 기기들
trait Device {
    fn is_enabled(&self) -> bool;
    fn enable(&mut self);
    fn disable(&mut self);
}

struct Tv {
    on: bool,
}
impl Device for Tv {
    fn is_enabled(&self) -> bool { self.on }
    fn enable(&mut self) { self.on = true; }
    fn disable(&mut self) { self.on = false; }
}

struct Radio {
    on: bool,
}
impl Device for Radio {
    fn is_enabled(&self) -> bool { self.on }
    fn enable(&mut self) { self.on = true; }
    fn disable(&mut self) { self.on = false; }
}

// 2. 추상층 (Abstraction)
struct Remote {
    // 디바이스 트레이트 객체를 소유하여 두 계층을 연결합니다.
    device: Box<dyn Device>,
}

impl Remote {
    fn new(device: Box<dyn Device>) -> Self {
        Remote { device }
    }

    // 기기의 전원 상태를 반전시키는 메서드를 구현하세요.
    // device의 상태를 확인(is_enabled)한 후 켜져 있으면 끄고(disable), 꺼져 있으면 켭니다(enable).
    fn toggle_power(&mut self) {
        // 이 부분을 구현하세요.
    }

    // 테스트 검증을 위한 헬퍼 메서드
    fn is_device_on(&self) -> bool {
        self.device.is_enabled()
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_bridge_pattern_tv() {
        let tv = Box::new(Tv { on: false });
        let mut remote = Remote::new(tv);

        assert_eq!(remote.is_device_on(), false);
        remote.toggle_power();
        assert_eq!(remote.is_device_on(), true);
        remote.toggle_power();
        assert_eq!(remote.is_device_on(), false);
    }

    #[test]
    fn test_bridge_pattern_radio() {
        let radio = Box::new(Radio { on: true });
        let mut remote = Remote::new(radio);

        assert_eq!(remote.is_device_on(), true);
        remote.toggle_power();
        assert_eq!(remote.is_device_on(), false);
    }
}
```
