# 문제 2 (HashMap Entry API)

## [문제 2] HashMap의 Entry API를 활용한 카운터 구현

### 설명

자바에서 가방(Map)에 카운트를 누적할 때 `map.put(key, map.getOrDefault(key, 0) + 1)` 또는 `computeIfAbsent`를 사용하곤 합니다.

Rust에서는 **`Entry` API**를 사용하여 이 작업을 단 한 줄로 안전하고 효율적으로 처리합니다.

단어 리스트를 받아 각 단어가 몇 번 등장했는지 세어주는 빈도수 맵을 완성하세요.

- `entry(word)`를 통해 해당 키의 단어가 맵에 존재하는지 확인합니다.
- 존재하지 않는다면 `.or_insert(0)`를 통해 기본값 0을 주입하고, 그 값의 가변 참조를 얻어와 1을 증가시킵니다.

### 템플릿 코드

```rust
use std::collections::HashMap;

fn count_words(words: Vec<String>) -> HashMap<String, u32> {
    let mut frequencies = HashMap::new();

    for word in words {
        // 이 부분을 구현하세요.
        // entry()와 or_insert()를 체이닝하여 구현해 보세요.
    }

    frequencies
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_hashmap_entry() {
        let words = vec![
            "apple".to_string(),
            "banana".to_string(),
            "apple".to_string(),
        ];

        let result = count_words(words);

        assert_eq!(result.get("apple"), Some(&2));
        assert_eq!(result.get("banana"), Some(&1));
        assert_eq!(result.get("orange"), None);
    }
}
```
