# 어댑터 (Adapter)

- **목표:** 기존에 정의된 타깃 인터페이스를 따르지 않는 서드파티나 레거시 코드를, 현재 시스템의 인터페이스에 맞춰 변환해 주는 어댑터 패턴을 구현합니다.
- **조건:** Rust에서는 호환되지 않는 구조체를 내부에 필드로 소유(Composition)하는 방식으로 어댑터 구조체를 정의하고, 타깃 인터페이스 역할을 하는 `trait`를 구현하여 내부 객체의 메서드를 호출해 주는 방식으로 구현합니다.

### [연습 문제] 어댑터 패턴 구현

현재 시스템은 사각형의 면적을 계산하는 `Shape` 인터페이스를 사용하고 있습니다. 새로 도입된 서드파티 라이브러리의 원형 구조체 `LegacyRoundPeg`를 현재 시스템의 `Shape` 인터페이스에 맞춰 작동하도록 변환하는 어댑터를 구현하세요.

1. `Shape` 트레이트는 `get_area` 메서드를 가집니다.
2. `LegacyRoundPeg` 구조체는 `radius` 필드와 이를 반환하는 `get_radius` 메서드를 가집니다. (이 코드는 수정할 수 없습니다.)
3. `SquarePegAdapter` 구조체는 내부에 `LegacyRoundPeg`를 소유하고 `Shape` 트레이트를 구현하여, 원의 반지름을 기반으로 마치 사각형인 것처럼 면적($\text{radius} \times \text{radius} \times 4$)을 계산하여 반환합니다.

```rust
// 1. 현재 시스템의 타깃 인터페이스
trait Shape {
    fn get_area(&self) -> f64;
}

// 2. 호환되지 않는 서드파티/레거시 컴포넌트 (Adaptee)
struct LegacyRoundPeg {
    radius: f64,
}

impl LegacyRoundPeg {
    fn new(radius: f64) -> Self {
        LegacyRoundPeg { radius }
    }

    fn get_radius(&self) -> f64 {
        self.radius
    }
}

// 3. 어댑터 구조체 (Adapter)
struct SquarePegAdapter {
    round_peg: LegacyRoundPeg,
}

impl SquarePegAdapter {
    fn new(round_peg: LegacyRoundPeg) -> Self {
        SquarePegAdapter { round_peg }
    }
}

// SquarePegAdapter에 Shape 트레이트를 구현하세요.
// get_area 메서드는 내부 round_peg의 지름을 한 변으로 하는 사각형의 면적을 구합니다.
// 즉, 공식은 (반지름 * 2.0)의 제곱, 혹은 (반지름 * 반지름 * 4.0) 입니다.
impl Shape for SquarePegAdapter {
    // 이 부분을 구현하세요.
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_adapter_pattern() {
        let round_peg = LegacyRoundPeg::new(5.0);
        let adapter = SquarePegAdapter::new(round_peg);

        // 반지름이 5.0인 원의 외접 사각형 한 변의 길이는 10.0이 되므로 면적은 100.0입니다.
        assert_eq!(adapter.get_area(), 100.0);
    }
}
```
