# 스택으로 큐 구현 / 큐로 스택 구현:

- **목표:** 구조체(`struct`)와 메서드(`impl`)를 설계하고, 두 개의 벡터(`Vec<i32>`) 간에 데이터를 이동시키며 스택으로 큐의 동작(FIFO)을, 큐로 스택의 동작(LIFO)을 시뮬레이션합니다.
- **조건:** 구조체의 상태를 변경해야 하므로 메서드에서 `&mut self`를 올바르게 사용하고, 벡터의 `pop()`이 반환하는 `Option` 타입을 안전하게 처리해야 합니다.
