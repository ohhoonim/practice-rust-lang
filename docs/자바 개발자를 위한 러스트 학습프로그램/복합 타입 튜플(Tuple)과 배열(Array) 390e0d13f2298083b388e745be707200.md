# 복합 타입: 튜플(Tuple)과 배열(Array)

- **요약 목록**
    - **튜플(`Tuple`)**: **서로 다른 타입**의 값들을 하나로 묶을 수 있으며, 고정된 길이를 가집니다. 점(`.`)과 인덱스로 요소에 접근합니다.
    - **배열(`Array`)**: **같은 타입**의 값들만 묶을 수 있으며, 고정된 길이를 가집니다. 대괄호(`[]`)와 인덱스로 요소에 접근합니다.

## 튜플 (Tuple)

튜플은 다양한 타입의 여러 값들을 하나의 복합 타입으로 그룹화하는 일반적인 방법입니다. 튜플은 한 번 선언하면 **그 크기를 키우거나 줄일 수 없습니다.**

```rust
let tup: (i32, f64, u8) = (500, 6.4, 1);
```

### 요소에 접근하는 방법

1. **구조 분해 (Destructuring)**: 튜플의 값을 개별 변수로 나누어 담을 수 있습니다.
2. **점(`.`) 연산자**: 인덱스 번호를 사용하여 직접 접근합니다. 인덱스는 0부터 시작합니다.

```rust
let tup = (500, 6.4, 1);

// 1. 구조 분해
let (x, y, z) = tup;

// 2. 점 연산자 사용
let five_hundred = tup.0;
let six_point_four = tup.1;
```

## 배열 (Array)

배열은 튜플과 달리 **모든 요소가 반드시 같은 타입**이어야 합니다. Rust의 배열은 고정된 길이를 가지므로, 한 번 선언하면 크기를 변경할 수 없습니다. (크기가 유동적인 컬렉션을 원한다면 추후 배울 `Vector`를 사용해야 합니다.)

```rust
let a = [1, 2, 3, 4, 5];
let b: [i32; 5] = [1, 2, 3, 4, 5]; // [타입; 길이] 형식으로 선언
```

### 동일한 값으로 초기화하기

같은 값으로 채워진 배열을 만들 때는 대괄호 안에 `[초기값; 길이]` 형태로 쉽게 선언할 수 있습니다.

```rust
let a = [3; 5]; // [3, 3, 3, 3, 3]과 동일
```

### 요소에 접근하는 방법

대괄호(`[]`)와 인덱스를 사용하여 값에 접근합니다. 인덱스는 0부터 시작합니다.

```rust
let a = [1, 2, 3, 4, 5];
let first = a[0];
let second = a[1];
```

> **주의**: 배열의 크기를 벗어난 인덱스(예: `a[10]`)에 접근하려고 하면, Rust는 컴파일 시점 혹은 실행 시점(Runtime)에 에러(`panic`)를 발생시켜 메모리 안전성을 보장합니다.
