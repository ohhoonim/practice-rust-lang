# 문제 2 (BufReader를 이용한 파일 읽기)

## [문제 2] BufReader를 이용한 한 줄씩 파일 읽기

### 설명

자바의 `new BufferedReader(new FileReader(file))` 구조처럼, 대용량 파일 처리를 위해 메모리 버퍼를 대동하여 데이터를 라인 단위로 파싱하는 훈련입니다.

`Read` 트레이트를 구현한 가상의 스트림(또는 파일)을 받아 각 줄을 읽고, `Option`에 담긴 문자열 벡터로 수집하세요.

- `std::io::BufRead` 트레이트의 `.lines()` 메서드를 활용하면 각 라인을 이터레이터 형태로 가져올 수 있습니다.
- 각 라인을 읽을 때 발생할 수 있는 에러는 `?` 연산자로 처리하여 함수 상위로 던지세요.

### 템플릿 코드

```rust
use std::io::{self, BufRead, BufReader, Read};

fn read_lines_from_stream<R: Read>(stream: R) -> io::Result<Vec<String>> {
    let reader = BufReader::new(stream);
    let mut lines_vec = Vec::new();

    // reader.lines()를 순회하며 각 라인의 텍스트를 lines_vec에 추가하세요.
```
